# Anomalo：Agent をブラウザから現実世界へ

Anomalo は、私の個人 AI エンジニアリングラボです。イベント駆動型 FastAPI Agent Host と Vue コントロールパネルを中心に、Agent Harness の実験、StackChan ロボット制御、個人株式リサーチを、一つの観測可能で実際に動くシステムへ統合しています。

## このプロジェクトを作る理由

Agent Harness 技術の進歩は非常に速く、新しいアーキテクチャ、ツール、実践方法がほぼ毎日のように登場しています。こうした新技術を学ぶ最良の方法は、ドキュメントを読んだりフレームワークを比較したりするだけでなく、自分の手で一通り作ってみることだと考えています。

だからこそ、新しい Agent 技術を、記事や孤立したデモ、フレームワーク比較だけで終わらせたくありません。ストリーミングランタイム、Tool Calling、コンテキスト構築、Prompt Profile、Memory、Skills、MCP、サンドボックス実行、人間による承認フローは、実際に動くシステムとして接続して初めて、その境界とコストが見えてきます。

Anomalo では Agent の身体性も探求します。Buddy Bridge を通じて StackChan 形式のデバイスへ接続し、シリアルまたは TCP 通信、タッチイベント、状態変化、音声ターン、低頻度の視覚、動作コマンドを扱えます。ブラウザ内の Agent は文字を返すだけでなく、現実環境を感知し、影響を与え始めます。

三つ目の方向は個人株式リサーチです。リポジトリには、データアダプター、テクニカルな証拠、ランキング、レポートを備えた、決定論的でテスト可能な市場分析エンジンがあります。AI の役割は、市場背景を説明し、セットアップを調査し、研究を整理することであり、不透明な自動売買システムになることではありません。

## 現在の実装

- WebSocket または REST で Agent のライフサイクル、メッセージ、ツールイベントをストリーミング。
- OpenRouter を標準とする OpenAI SDK 互換クライアントとローカル Mock モード。
- 実行時に Prompt Profile、Memory、Python Skills、MCP Server をロード。
- チャット、コンテキスト確認、Buddy 制御、株式分析を一つの Vue パネルに統合。
- STT、TTS、ローカル音声対話、StackChan デバイスブリッジをサポート。
- Mock データまたは Futu OpenD による決定論的な株式分析。
- 独立した Python サンドボックスサービスと Apple Container のビルド・デプロイに対応。

## Technical Specifications

FastAPI / Vue 3 / Vite / OpenRouter / MCP / Python 3.12 / WebSocket / Apple Container

## プロジェクト状態

Anomalo は現在 v0.1 として活発に開発中です。信頼できるネットワークで動かす個人実験であり、強化されたマルチユーザーサービスではありません。次の段階では、Agent、実体デバイス、リサーチツールの協調境界をさらに明確にし、すべてのツール呼び出し、判断、人間の承認を観測可能かつ説明可能な状態に保ちます。
