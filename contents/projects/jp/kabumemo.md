# Kabumemo

Vibecoding の試みとして、手書きコードを使わず GPT-5/Codex への指示だけで生成した個人向け株式取引記録サイト。最小コストで個別要件を満たすことを目標としている。

## Technical Specifications

GPT, Static Site, JSON

## Classification Tags

Vibe Coding / TypeScript / Python

## Background

このプロジェクトは自律型エージェントシステムの継続的な進化を示すものであり、人工知能応用における工学的な卓越性と革新への取り組みを反映している。
