# Agent Applications on Azure AI Foundry

Built on Azure AI Foundry, this work covered capabilities such as multi-agent collaboration, MCP invocation, and RAG retrieval.

## Overview

1. Practice 1: MCP Invocation

I developed an app with FastAPI so that AI could write Python scripts. Through the ███████ MCP protocol, the Agent can now execute these scripts by itself and complete ███████ business workflows with full automation.

2. Practice 2: Multi-Agent Collaborative Retrieval

Using Azure AI Foundry, I built a multi-expert collaboration pattern:

- Expert A: turns vague user needs and ██ identifiers into professional search instructions.
- Expert B: searches the web for reference material.
- Expert C: aggregates, structures, and organizes the information, and requests more searches when something is missing.

This approach improved the information retrieval quality for █████.

3. Practice 3: Knowledge Base (RAG) and Output QA

I added a RAG knowledge base to the █████ feature. Before answering, the Agent first checks the base for know-how to optimize the prompt. The key part was designing an inter-agent evaluation mechanism that automatically checks whether the generated answer truly referenced that knowledge. In the end, it improved the accuracy of the █████ feature.
