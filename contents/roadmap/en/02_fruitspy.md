# FruitSpy

This project is mainly for keeping an eye on the Mac mini server at home. The initial idea was very simple: I just wanted to make a page where I could casually check CPU and memory usage. Later, for security reasons, I added a "health check" function to it, which can pull out all npm, pip, and uv package information and versions on the development machine with one click, so that I will not stay unaware even if a supply chain attack happens. Besides viewing system status, it can also monitor the running status of Docker containers. If I want to do deeper management, one click can take me directly to Portainer. This project is mainly developed as needed for now, and there is no long-term plan yet. It is a typical "add whatever is missing" tool, and whenever I discover a new need, I can stuff more code into it at any time.
